How do I make my homepage colorless?

It is really simple! Ask your webmaster to include the following code before the </body> closing tag:

<script src="https://www.blacklivesmatter.support/js/blms.js"></script>

Your homepage will become colorless only during these 3 days.

There is nothing else to do!

Do you have a plugin for WordPress?

Of course! First, download the plugin.

Then, login your dashboard and goto Plugins > Add new. Install the plugin and activate it!

How can I validate taht the code will work?

Simply add ?blms_simulation at the end of your homepage URL in your browser and you should see your homepage colorless!

Does it work on any website?

It should. That being said, we are using the CSS filter instruction to convert your website to black and white. This might not work in every case. For instance, there are limitaions in Microsoft Edge (and Internet explorer!) if you are using z-index. You might have to add more code to make your homepage 100% colorless.
