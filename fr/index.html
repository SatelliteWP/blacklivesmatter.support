Comment enlever les couleurs de ma page d'accueil?

C'est vraiment simple! Demandez à votre webmestre d'inclure le code suivant avant la balise de fermeture </body> :

<script defer src="https://blacklivesmatter.support/js/blms.js"></script>

Votre page d'accueil verra ses couleurs retirées seulement pour les trois (03) jours mentionnés.

Il n'y a rien d'autre à faire!

Avez-vous une extension pour WordPress?

Bien sûr! Premièrement, téléchargez l'extension.

Puis, à partir de votre tableau de bord WordPress, accédez aux menu Extensions > Ajouter nouveau. Installez l'extension et activez-la!

Après l'activation, vous pouvez accéder à la page de configuration de l'extension:

Methode 1 : Dans le menu Extensions > Extensions installées, cliquez sur le lien "Settings" (Réglages) tel qu'illustré :

Dans la page de configuration, vous pouvez :

  • Activer ou désactiver le mode débogage avec l'option "Debug mode" (par défaut, le mode débogage désactivé)
  • Déterminer l'emplacement du badge sur la page d'accueil (par défaut, le badge est positionné "En bas à droite").
  • Inscrire le lien de vos pages d'accueil en anglais et en français.
  • Prévisualiser votre page d'accueil sans couleur

    • Une fois les modifications complétées, n'oubliez pas de sauvegarder vos changements!

    Comment puis-je valider que le code fonctionnera?

    Seulement à ajouter ?blms_simulation à la fin de l'adresse de la page d'accueil de votre site dans votre navigateur et vous pourrez voir votre page d'accueil sans couleur!

    Est-ce possible d'avoir plus de contrôle sur l'intégration du mode sans couleur?

    Oui! Plusieurs options sont disponibles.

    Vous pouvez insérer les codes ci-dessous à l'intérieur d'une balise <script>. Placez ensuite l'ensemble juste avant le code ci-dessus

    Pour activer le débogage :

      var blms_debug = true;
    Note: Par défault, la valeur du blms_debug est à false et le débogage désactivé.

    Pour positionner le badge sur la page d'accueil :

      var blms_badge_location = "bottomright";
    Note: blms_badge_location peut prendre la valeur "topleft", "topright", "bottomleft" ou par défaut "bottomright"

    Forcer la page actuelle à s'afficher sans couleur (le tout doit être fait manuellement sur chaque page) :

      var blms_force = false;

    Spécifier toutes les pages et leur langue que vous souhaitez afficher sans couleur :

      // Langues supportées : fr, en.
      var blms_pages = {'/' : 'en', '/fr/' : 'fr' };
    Note: Le chemin (e.g. '/') doit être exactement ce qui se trouve dans l'URL.

    Est-ce que ça fonctionne sur tous les sites web?

    Ça devrait. Ceci étant dit, nous utilisons un filtre CSS pour convertir votre site au noir et blanc. Il est possible que ça ne fonctionne pas dans tous les cas. Par exemple, il existe certaines limitations avec Microsoft Edge si votre site utilise des z-index. Les filtres ne sont pas supportés sur Internet Explorer 11! Vous pourriez avoir à ajouter un peu de code pour que 100% des couleurs soient retirées de votre page d'accueil.
